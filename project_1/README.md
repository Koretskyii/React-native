# Лабораторна робота №1: Базовий мобільний застосунок
## 🎯 Мета
Ознайомлення з базовими компонентами React Native, налаштуванням середовища Expo та створенням найпростішої стекової навігації.
## 🛠 Що було використано
- **React Native & Expo:** Базові компоненти `View`, `Text`, `TouchableOpacity`, `StyleSheet`.
- **React Navigation (Stack):** Для реалізації переходів між сторінками.
## 📱 Опис застосунку
Створено базовий багатоекранний застосунок. Він включає Головний екран (Home), екран Профілю користувача (Profile) та екран Налаштувань (Settings). Реалізовано переходи між цими сторінками за допомогою кнопок, а також базову стилізацію елементів інтерфейсу.
