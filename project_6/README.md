# Лабораторна робота №6: Firebase Auth та Firestore
## 🎯 Мета
Інтеграція хмарних сервісів Google Firebase для повноцінної автентифікації та збереження даних у віддаленій базі.
## 🛠 Що було використано
- **BaaS:** Firebase (`firebase/auth`, `firebase/firestore`).
- **Функціонал:** `signInWithEmailAndPassword`, `updateDoc`, `reauthenticateWithCredential`.
## 📱 Опис застосунку
Фінальний додаток містить реальну систему акаунтів. Користувач може зареєструватися, скинути пароль або зайти у свій профіль. У профілі можна вказати своє ім'я, вік та місто — ці дані зберігаються у Firestore. У базі налаштовані Security Rules, що забороняють користувачам читати чужі дані.
