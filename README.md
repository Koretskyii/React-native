## 💡 Загальний висновок
В ході виконання цього циклу лабораторних робіт було здобуто комплексні навички розробки мобільних застосунків за допомогою React Native та фреймворку Expo. 
Вивчено проєктування архітектури додатків, опрацьовано різні типи навігації (React Navigation та Expo Router), управління глобальним станом через React Context, 
взаємодія з апаратними можливостями пристрою (файлова система, жести) та інтеграція хмарних сервісів Firebase для повноцінної роботи з даними та автентифікацією користувачів.
